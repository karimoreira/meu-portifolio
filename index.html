Alguns dos meus projetos:

Projeto #1

Clima Tempo com OpenWeather API

O "Clima Tempo com OpenWeather API" é um projeto que utiliza a API do OpenWeather para fornecer informações detalhadas sobre o clima em tempo real. O objetivo é permitir que os usuários consultem dados meteorológicos, como temperatura, condições climáticas e previsões para diferentes locais.

HTML, CSS, JavaScript, APIs

Projeto #2

Memory Game - Rick and Morty

O "Memory Game - Rick and Morty" é um jogo de memória baseado na minha série favorita: Rick and Morty. O jogo desafia os usuários a encontrar pares de cartas correspondentes, usando personagens da série. O jogo oferece uma experiência divertida e interativa enquanto testa a memória dos jogadores.

HTML, CSS, JS

Projeto #3

Criação de uma api de provas e certificações

Neste projeto desenvolvi pela primeira vez uma API para gerenciar e distribuir provas e certificações de alunos, permitindo a criação, atualização e consulta de informações com precisão para uma melhor experiência dos usuários.

Java, Docker + PostgreSQL, Postman
